# Jack Gustafssons Portfolio

Hej! Detta är min personliga portfolio där jag visar upp mina projekt inom frontend-utveckling.

## Om projektet

Denna webbplats är byggd med HTML och CSS. Den visar mina projekt, om mig och kontaktinformation.

- Mobilvänlig (mobile-first)
- Responsiv design med flexbox och grid
- Enkel, användarvänlig layout

## Funktioner

- Hemsida med välkomsttext och kort presentation
- Projekt-sida med exempel på webbsidor och spel
- Om mig-sida med bakgrund och erfarenheter
- Kontakt-sida med sociala länkar

## Teknologier

- HTML5
- CSS (inklusive flexbox och media queries)
- Font Awesome för sociala ikoner
